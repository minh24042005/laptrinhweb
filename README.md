Chủ đề 1: Hệ thống Quản lý Sinh viên Đơn giản
• Mô tả: Ứng dụng cho phép quản lý thông tin sinh viên.
• Các thực thể (MySQL): SinhVien (Mã SV, Tên SV, Ngày sinh, Giới tính, Lớp).
• Chức năng chính:
o Thêm, Sửa, Xóa, Xem danh sách sinh viên.
o Tìm kiếm sinh viên theo tên hoặc mã.
o Đăng nhập/đăng xuất cho quản trị viên.
